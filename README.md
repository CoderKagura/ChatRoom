### 实现步骤
1. 用JFrame制作客户端和服务端的界面
2. 使用Socket网络编程TCP协议完成数据传输
3. 监听`发送`按钮的点击事件，实现点击发送按钮发送消息
4. 监听键盘点击事件，实现点击回车键发送消息
5. 将服务器端口号、客户端ip地址、客户端端口号写入配置文件
